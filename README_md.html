    Alimento¶ ↑

    Prueba de creación de una gema con bundler. Programa para calcular el impacto ambiental diario de los alimentos. Se ha implementado una lista doblemente enlazada para almacenar alimentos. Creación de la clase PlatosN y PlatosA, que almacenan listas de alimentos, así como el % de proteínas, hidratos, etc. de cada plato.

    Daniel Nuez Wehbe, alu0100881165

    Uso¶ ↑

    'rake spec' para ejecutar los tests. 'bin/setup' para instalar las dependencias. 'bin/console' para una consola interactiva. 'bundle exec rake install' para instalar la gema en la máquina local. 'bundle exec rake release' (una vez modificado version.rb) para crear una nueva versión.
